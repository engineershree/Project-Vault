This project is designed to **streamline project management for students and faculty** by providing a centralized platform to manage academic projects efficiently.

It stores both **past and ongoing project materials**, allowing students to upload content directly from their **GitHub repositories**. The system integrates with the **GitHub API** to ensure secure and reliable access to repositories.

The project also provides a **dedicated section for students to submit project updates, progress reports, and relevant files**, making it easier for faculty to track progress and review work in an organized manner.
